# WeatherNow – Real-Time Weather Forecast Website

A responsive weather website that provides real-time weather updates, hourly forecasts, and city-based climate information using a weather API.

## Features
- Live weather data
- Search weather by city
- Temperature, humidity, wind speed
- 5-day forecast
- Responsive design for mobile and desktop
- Clean UI with dark/light mode

## Built With
- HTML
- CSS
- JavaScript
- OpenWeather API (or your API)
